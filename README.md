# AI-Driving-Test

An AI-powered driver behavior analysis system built using OpenCV that evaluates driving performance from dashcam footage and generates a comprehensive driving skill report.

---

## Overview

Driving behavior is one of the major factors affecting road safety. This project analyzes recorded driving videos and evaluates various aspects of driving using computer vision techniques.

The system detects driving events, monitors lane discipline, estimates driver performance, and generates a final driving report with scores and recommendations.

---

## Features

* Lane Detection
* Traffic Sign Detection
* Driving Event Detection
* Driver Skill Scoring
* Automatic Report Generation
* Video Processing Pipeline
